  2. COLLAPSIBLE CARD FIX - PROPER RESIZING
═══════════════════════════════════════════════════════════════

ISSUE: When clicking triangle to collapse cards, the card stayed
       the same size even though content was hidden

FIX APPLIED:

1. Updated .card-content base styles:
   • Removed fixed max-height
   • Added proper transitions (0.4s ease)
   • Non-collapsible content: max-height: none

2. Collapsible content classes:
   • .personal-info-content
   • .application-details-content
   • .financial-info-content
   • .activity-content
   
   Collapsed state:
   • max-height: 0
   • padding: 0 20px
   • opacity: 0
   
   Expanded state:
   • max-height: 400-600px
   • padding: 20px
   • opacity: 1

3. Updated .info-cards-grid:
   • Added align-items: start
   • Allows cards to have different heights
   • Cards can shrink/grow independently

4. Updated .premium-card:
   • height: auto
   • min-height: fit-content
   • Smooth transitions (0.4s ease)

RESULT:
  ✅ Collapsed: Card shrinks to header only (~60px height)
  ✅ Expanded: Card grows to show all content
  ✅ Smooth animation with proper easing
  ✅ Triangle icon rotates 180° when expanded
  ✅ Other cards maintain their size independently
