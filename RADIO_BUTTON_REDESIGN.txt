CSS Features Used:
  ✓ CSS :has() selector for parent styling

  ✓ Transform and box-shadow transitions

  • All transitions: 0.3s ease

Accessibility:

  ✓ Keyboard accessible (hidden radio still works)
  ✓ Clear visual feedback on selection
  ✓ High contrast colors
  ✓ Proper cursor: pointer
