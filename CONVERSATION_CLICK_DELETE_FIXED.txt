═══════════════════════════════════════════════════════════════
  HOW IT WORKS NOW
═══════════════════════════════════════════════════════════════

CLICKING CONVERSATION:
1. User clicks conversation
2. Check if `hasMoved` flag is false
3. Check if wrapper is not swiped
4. If both true → Navigate to conversation
5. If either false → Prevent navigation

SWIPING TO DELETE:
1. User touches/clicks and holds
2. Moves left more than 5px → `hasMoved = true`
3. Moves more than 50px → Show delete button
4. Release → Delete button stays visible
5. Click delete → Modal shows
6. Confirm → Conversation deleted

PREVENTING CONFLICTS:
→ Swipe detection prevents accidental clicks
→ Click detection prevents accidental swipes
→ 5px threshold for movement detection
→ Separate flags for touch and mouse events
