IF YOU DON'T SEE THE CHANGES:
1. Make sure you restarted the application (Ctrl+C then python run.py)
2. Hard refresh your browser (Ctrl+Shift+R)
3. Clear browser cache completely
4. Try a different browser or incognito mode

IF DELETE BUTTON DOESN'T WORK:
1. Check browser console for errors (F12)
2. Make sure you're logged in as admin
3. Confirm the CSRF token is working

IF ID DOCUMENT DOESN'T SHOW:
1. Make sure the application has an uploaded document
2. Check that the file exists in: app/static/uploads/documents/
3. Verify the application.id_document_path is not None
